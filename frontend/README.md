# ATS AI Interviewer - Frontend

React + TypeScript + Material-UI frontend for the ATS application.

## Setup

```bash
npm install
npm run dev
```

## Features

- Role-based dashboards (Admin, Recruiter, Candidate)
- JWT authentication
- Material-UI components
- Responsive design
- Protected routes

## Pages

### Admin
- Dashboard with statistics
- View all candidates
- View all recruiters  
- AI interview analysis with Q&A

### Recruiter
- Dashboard
- Post new jobs
- Manage job postings
- View applications

### Candidate
- Dashboard
- Browse jobs
- Apply for jobs
- Upload resume
- Complete AI interviews

## Tech Stack

- React 18
- TypeScript
- Material-UI (MUI)
- React Router v6
- Axios
- Vite
